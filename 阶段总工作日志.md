### 分析设计
#### 1. 长期耦合（跨区域）

主检验包括：
- `total_annual_area_km2_anomaly_z` vs `glambie_combined_mwe_anomaly`
- `mean_ratio_anomaly_z` vs `glambie_combined_mwe_anomaly`

- `total_annual_area_km2` vs `glambie_combined_mwe_cumulative`
- `zero_ratio_share_anomaly_z` vs `glambie_combined_mwe_anomaly`

#### 3. 区域逐区相关检验
对每个区域单独计算：
- Pearson
- Spearman

便于查看区域异质性与主效应是否由个别区域驱动。

### 样本与缺失处理
- 原 panel：425 行
- GlaMBIE annual 缺失：17 行
- 缺失年份全部为：`2024`
- 用于正式 GlaMBIE 统计检验的共享样本：`408` 行

### 关键初步结果
#### A. 长期耦合结果
当前第一轮长期跨区域结果并不强：

- `lake_area_change_2000_2023_km2` vs `glambie_combined_mwe_cumulative_2023`
  - Pearson = `-0.0422`
  - Spearman = `0.1348`
  - OLS slope = `-1.7337`
  - `p ≈ 0.872`

- `lake_area_change_2000_2023_km2` vs `glambie_combined_gt_cumulative_2023`
  - Pearson = `0.1941`
  - Spearman = `0.0686`
  - OLS slope = `0.1245`
  - `p ≈ 0.455`

说明：
- 在当前这版最简单的“跨区域长期累计量”检验下，还没有看到强而清晰的全球长期耦合信号。
- 这一步更像是 baseline check，而不是最终机制结论。

#### B. 年际 anomaly 主结果
当前第一轮 pooled 年际检验已经出现较明确的负相关信号：

- `total_annual_area_km2_anomaly_z` vs `glambie_combined_mwe_anomaly`
  - Pearson = `-0.2480`
  - OLS slope = `-0.4251`
  - `p ≈ 3.91e-07`
  - `R² ≈ 0.0615`

- `mean_ratio_anomaly_z` vs `glambie_combined_mwe_anomaly`
  - Pearson = `-0.1963`
  - OLS slope = `-0.3271`
  - `p ≈ 6.56e-05`
  - `R² ≈ 0.0385`

- `zero_ratio_share_anomaly_z` vs `glambie_combined_mwe_anomaly`
  - Pearson = `0.2678`
  - OLS slope = `0.4727`
  - `p ≈ 3.93e-08`
  - `R² ≈ 0.0717`

这说明：
- 年际 glacier mass-balance anomaly 与 lake anomalies 之间已经出现可检出的统计联系
- 方向上表现为：更负的质量平衡异常对应更高的 lake-area anomaly / ratio anomaly，以及更低的 zero-ratio state
- 这个结果为后续“极端暖季放大”和“双时间尺度机制”提供了第一层定量基础

#### C. rolling / cumulative 的第一轮结果
- `usable_total_area_change_km2` vs `glambie_combined_mwe_rolling3`
  - Pearson = `0.0187`
  - `p ≈ 0.712`
- `usable_total_area_change_km2` vs `glambie_combined_mwe_rolling5`
  - Pearson = `0.0103`
  - `p ≈ 0.839`
- `usable_total_area_change_km2` vs `glambie_combined_mwe_cumulative`
  - Pearson = `0.0049`
  - `p ≈ 0.923`
- `total_annual_area_km2` vs `glambie_combined_mwe_cumulative`
  - Pearson = `0.0458`
  - `p ≈ 0.356`

说明：
- 这版 rolling / cumulative 的 pooled 简单检验暂时不强
- 初步判断是：当前较清楚的信号主要出现在 anomaly 层，而不是 rolling 或 cumulative 的简单 pooled 形式中
- 后续更适合对长期层做“region-level summarized metric redesign”，而不是直接继续堆叠同一形式的 cumulative pooled test

#### D. 区域异质性初步印象
从 `regionwise_correlations.csv` 可见，年际 anomaly 关联在不同区域之间差异明显。

在 `mean_ratio_anomaly_z` vs `glambie_combined_mwe_anomaly` 上，负相关较明显的区域包括：
- `alaska`
- `caucasus_middle_east`
- `greenland_periphery`
- `southern_andes`

在 `total_annual_area_km2_anomaly_z` vs `glambie_combined_mwe_anomaly` 上，负相关更明显的区域包括：
- `alaska`
- `arctic_canada_north`
- `caucasus_middle_east`
- `greenland_periphery`
- `scandinavia`
- `western_canada_us`

说明：
- pooled 主结果不是完全由单一区域驱动
- 但确实存在明显区域异质性
- 后续可以据此挑选主案例区与支持区

### 这一步的阶段判断
到当前为止，可以先得出一个谨慎但有价值的判断：

1. **长期累计质量变化 vs 长期湖泊面积变化** 的最简单跨区域检验，目前信号不强。

4. 目前最值得继续深挖的是：
   - 年际 anomaly 结果的稳健性

   - 后续再叠加 ERA5-Land 的暖季异常解释
